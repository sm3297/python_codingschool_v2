# 🐍 파이썬 플레이그라운드

> 중학교 1학년 학생들을 위한 게임형 파이썬 미션 학습 사이트

## 🚀 시작하기

### 설치

```bash
npm install
```

### 실행

```bash
npm run dev
```

또는

```bash
npm start
```

### 빌드

```bash
npm run build
```

## 📖 소개

파이썬 플레이그라운드는 8개의 스테이지를 순서대로 클리어하면서 파이썬을 배우는 미션형 학습 사이트입니다.

## 🔑 계정 정보

### 선생님 계정
- 아이디: `seongmin`
- 비밀번호: `1`

### 학생 계정
- 회원가입 후 사용

## 🛠️ 기술 스택

- React
- Vite
- JavaScript
- React Router
- Axios
- MockAPI
- CSS

## 📁 프로젝트 구조

```
src/
  api/          - MockAPI 통신 함수
  components/   - 재사용 컴포넌트
  data/         - 스테이지/미션 데이터
  pages/        - 페이지 컴포넌트
  styles/       - CSS 스타일
  utils/        - 유틸리티 함수
```
